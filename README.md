# 基于debian系统的Socks5协议安装脚本

## 功能展示：
```
--- SOCKS5 代理管理脚本1.0.0 日期：2025.07.24 ---
1. 安装 SOCKS5 代理
2. 卸载 SOCKS5 代理
3. 查看 SOCKS5 代理状态
4. 修改 SOCKS5 配置 (端口/用户/密码)
5. 重启 SOCKS5 服务
6. 退出
----------------------------
请输入您的选择 (1-6)：
```
## 一键脚本：
```
curl -L https://github.com/sssspc/Socks5-Person/releases/download/v1.0.0/socks5.sh -o socks5.sh && chmod +x socks5.sh &&  ./socks5.sh
```
